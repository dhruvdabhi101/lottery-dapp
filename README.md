Stack Deez

it is a lottery Dapp built with solidity, Nextjs, ChakraUI, Tailwind.

## Installation

```bash
npm install
```

```bash
npm run dev
```

## Features
- Normal user can stake their `0.000015 eth` to enter the lottery.
- Owner can start the lottery and end the lottery and restart the lottery.
